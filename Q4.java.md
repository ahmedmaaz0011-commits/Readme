Write a program to calculate and print the original selling price if the discounted selling
price and discount percentage is entered by the user in simple java.

## Copilot

```java
import java.util.Scanner;

public class OriginalSellingPriceCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Input: discounted selling price
        System.out.print("Enter the discounted selling price: ");
        double discountedPrice = scanner.nextDouble();

        // Input: discount percentage
        System.out.print("Enter the discount percentage: ");
        double discountPercent = scanner.nextDouble();

        // Calculate the original selling price
        double originalPrice = discountedPrice * 100 / (100 - discountPercent);

        // Output the result
        System.out.printf("The original selling price is: %.2f\n", originalPrice);
        
        scanner.close();
    }
}
```

**How it works:**
- The user enters the discounted selling price and the discount percentage.
- The original price is found using the formula:  
  `Original Price = (Discounted Price * 100) / (100 - Discount Percentage)`
- The result is printed to two decimal places.

**Sample Run:**
```
Enter the discounted selling price: 800
Enter the discount percentage: 20
The original selling price is: 1000.00
```
